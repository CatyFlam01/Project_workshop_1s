# Прогнозирование стоимости медийной рекламной кампании


## О проекте

Этот учебный проект направлен на решение бизнес-задачи сети магазинов **Food Mart**. Проблема заключается в неэффективном распределении бюджетов на медийные рекламные кампании из-за отсутствия точного прогноза стоимости привлечения клиента (Customer Acquisition Cost).

**Цель проекта:** Разработать модель машинного обучения, которая по характеристикам магазина и данным клиентов будет предсказывать стоимость привлечения одного клиента (`cost`) с минимальной ошибкой (RMSLE).


## Данные

Исходные данные для проекта можно скачать по следующим ссылкам:

*   **Основной датасет (Kaggle Competition):** [Playground Series S3E11](https://www.kaggle.com/competitions/playground-series-s3e11/data?select=train.csv)

*   **Дополнительный датасет:** [Media Campaign Cost Prediction](https://www.kaggle.com/datasets/gauravduttakiit/media-campaign-cost-prediction)

## Ход работы

Проект будет реализован по следующему плану:

1.  **Первичный анализ данных (EDA):** Анализ распределений, поиск пропусков и выбросов.  
2.  **Визуализация и гипотезы:** Создание графиков и формулировка гипотез.  
3.  **Предобработка данных:** Обработка выбросов, масштабирование, кодирование.  
4.  **Feature Engineering:** Создание и отбор новых признаков.  
5.  **Базовое моделирование:** Обучение и оценка базовых моделей (Linear Regression, Decision Tree, Random Forest).  
6.  **Углубленное моделирование:** Подбор гиперпараметров, ансамбли.  
7.  **Анализ модели:** Интерпретация результатов и важности признаков.  
8.  **Финализация модели:** Обучение финальной модели и создание прогнозов.  
9.  **Подготовка отчета и защита.**  


## Результаты

Ожидаемые результаты проекта:

1.  Аналитический отчет (EDA).  
2.  Обученные ML-модели с сравнением их эффективности.  
3.  Анализ важности признаков (Feature Importance).  
4.  Рабочий прототип для генерации прогнозов.
